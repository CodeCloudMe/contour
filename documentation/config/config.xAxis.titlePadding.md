#### **titlePadding** : {number}

*default: 4* 

The distance below the axis and [`chart.padding.bottom`](#config_config.chart.padding.bottom), in pixels, where the axis `title` is drawn.

**Example:**

	new Narwhal({
	    el: ".myLineChart",
	    xAxis: { title: 'Index', titlePadding: 40 }
	  })
	.cartesian()
	.line(data)
	.render()

*[Try it.](http://jsfiddle.net/forio/BJNd8/)*
